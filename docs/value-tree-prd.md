# AI Valuation Tree — MVP PRD

**작성일:** 2026.02.28 | **작성자:** 영진 | **버전:** 1.0

---

## 한 줄 정의

회사명을 입력하면 AI가 밸류에이션을 트리 구조로 생성하고, 노드를 클릭하면 하위 근거를 실시간으로 드릴다운할 수 있는 도구

## 타겟 유저

밸류에이션이 어려워서 확신 없이 투자하는 개인투자자 (1차 유저: 나 자신)

## 풀려는 문제

- 밸류에이션이 어렵다 → 확신 없이 매수 → 조금 오르면 팔고, 살 타이밍에 못 삼
- 근본 원인: 논리적 밸류에이션을 구조화하는 프레임이 없음
- 기존 도구(HTS, 매매일지, 스프레드시트)는 "기록"만 하지 "사고 구조"를 제공하지 않음

## 핵심 기능 (MVP)

1. **회사명 입력 → AI가 밸류에이션 트리 자동 생성**
   - AI가 tool(웹검색, 임상DB, 재무API 등)으로 데이터를 수집
   - 핵심 밸류 드라이버를 추출하여 트리 구조로 구성
   - 각 노드마다 논리적 근거(rationale)와 출처 포함

2. **노드 클릭 → 하위 트리 실시간 드릴다운**
   - 궁금한 노드를 클릭하면 AI가 추가 조사하여 하위 노드 생성
   - 예: "패스트트랙 지정" 클릭 → 패스트트랙의 현금 가치, 승인 확률 변화, 유사 사례 비교
   - 무한 깊이로 탐구 가능

3. **사용자가 가정 수정 가능**
   - 숫자 노드를 클릭하여 본인 판단으로 수정
   - 예: "성공확률 35%" → 본인 판단 "50%"로 변경

## 기술 스택 (예상)

- **프론트엔드:** 웹 (React 또는 Flutter Web)
- **백엔드:** AI 에이전트 오케스트레이션 + Tool 연동
- **Tools:** 웹검색, DART 공시, PubMed/임상DB, 재무제표 API 등
- **AI:** Claude API (밸류에이션 트리 생성 + 드릴다운 생성)

## 차별점

| 기존 도구 | AI Valuation Tree |
|-----------|-------------------|
| 기록 중심 (메모, 매매일지) | 사고 구조 제공 (트리 + 근거) |
| 정적 데이터 | 실시간 드릴다운 |
| 사용자가 다 해야 함 | AI가 초안 + 데이터 수집 |
| 한번 보고 끝 | 궁금한 만큼 깊이 탐구 |
