Value Tree

[대표자명] · 2026. 03 · 투자검토용

회사명을 입력하면 AI가 밸류에이션 트리를 만들어주는 웹 서비스입니다.
증권사 리포트처럼 숫자만 던지는 게 아니라, 왜 이 값이 나왔는지를 트리 구조로 보여줍니다.

왜 만들고 있는가

SR투자파트너스에서 투자 검토를 하면서 느낀 건, 기업가치 평가라는 게 결국 소수의 전문가만 할 수 있는 작업이라는 겁니다. 개인투자자 1,400만 명 시대인데, 정작 "이 회사가 비싼 건지 싼 건지"를 스스로 판단할 수 있는 도구가 없습니다.

ChatGPT한테 물어보면 그럴듯한 숫자는 나옵니다. 근데 그 숫자가 어디서 온 건지, 배수는 왜 10배인지 알 수가 없어요. 결국 근거를 추적할 수 있는 구조가 필요하다고 생각했습니다.

작동 방식

"삼성전자" 라고 치면 이런 일이 일어납니다:

  • 공공데이터포털·DART에서 실제 재무데이터 조회
  • Google Search로 최신 뉴스·리포트 실시간 수집
  • AI가 기업 특성에 맞는 방법론(SOTP, DCF 등) 선택
  • 모든 가정과 계산을 트리 구조로 시각화
  • "배수를 12배로 바꿔줘" 하면 즉시 재계산

한국·미국 주식 모두 지원. 분석 소요시간 약 90초.

기존 대안과의 차이

핵심은 "왜?"가 보인다는 겁니다.

  • 증권사 리포트 — 결론만 있고, 가정을 바꿔볼 수 없음
  • Bloomberg — 연 3천만원, 개인은 접근 불가
  • ChatGPT — 출처 불명, 숫자의 근거를 추적 못함
  • Value Tree — 모든 노드에 근거·출처·사고과정 표시, 직접 수정 가능

수익 모델 (구상 중)

분석 건수 기반 과금이 자연스럽다고 보고 있습니다. 무료로 월 2~3회 써보게 하고, 그 이상은 유료 전환.

더 먼 그림으로는 증권사·핀테크에 API로 제공하는 B2B도 생각하고 있지만, 우선은 개인 사용자 반응을 먼저 확인하려 합니다.

만드는 사람

[대표자명] — 1인 개발 중. 이전에 Flutter·React·Spring 기반으로 누적 570만 회원, 연 거래액 1,000억+ 규모의 서비스를 설계·운영했습니다. 앱스토어·플레이스토어 스포츠 카테고리 1, 2위를 찍어봤고, 지금은 SR투자파트너스에서 투자 심사를 하면서 이 서비스의 필요성을 직접 느꼈습니다.

개발부터 배포까지 혼자 빠르게 돌릴 수 있는 게 현 시점의 강점이라고 생각합니다.

김기사랩에 기대하는 것

아직 투자 규모를 특정하기보다는, 제품을 같이 봐주실 분들이 필요합니다. 로직 고도화와 베타 운영에 필요한 초기 자금(AI API 비용 + 운영비)과 함께, 김기사랩 네트워크를 통한 초기 사용자 확보·피드백에 도움을 받고 싶습니다.
